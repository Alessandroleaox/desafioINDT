Olá,

A aplicação apresentada possui um desenvolvimento na linguagem HTML para criação da estruturação do hipertexto e utilizando o fremework Bootstrap que nos auxiliou em algum componentes de tela. O Java Script (JS) usamos para consumir a API disponibilizada.
Optamos por usar as tecnologias mais simples para facilitar a compreensão dos que estarão visualizando o condigo.

o Código está totalmente comentado para que cada elemento e função seja compreendida assim que avaliarem o código.

*Instruções Básico de Uso*
 
No primeiro momento, ao clicar no botão listar produtos, ele demorá 3 a 5 segundo para carregar.
Todas as vezes que efetuar uma modificação, atualize a pagina, clicando F5.
